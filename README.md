# gcc五分钟系列

在https://github.com/lexdene/gcc_five_minute上进行修改

## 简介

* 通过大量简单的例子介绍 `gcc` 的基本用法
* 每节内容简短，控制在5分钟以内
* 讲的是一些最基础的用法

## 关于版本
写本系列文章时，`gcc`的版本是4.6.2。
本系列文章参照此版本进行介绍。

如果`gcc`的最新版本的参数或用法发生更改，
本系列文章可能不会及时更新。
所以请您在正式使用前 *务必* 确认您的`gcc`版本并参考`gcc`的说明文档。
